Relational Properties

Relations

In Discrete Mathematics, a relation is a mathematical concept that defines a connection between elements of one or more sets. Formally, a relation R from set A to set B is a subset of the Cartesian product A Ã— B. In other words, a relation R consists of ordered pairs (a, b) where "a" is an element from set A, and "b" is an element from set B.

For example, let's define a relation R from set A = {1, 2, 3} to set B = {4, 5, 6} where each element in set A is related to its square in set B: R={(1,1),(2,4),(3,9)}

In this relation, (1, 1) means that 1 is related to 1, (2, 4) means that 2 is related to 4, and (3, 9) means that 3 is related to 9.

Properties of Relations

To understand the behavior and nature of relations in discrete mathematics, we ascribe the properties of reflexive, symmetric, and transitive to relations. It is the presence or absence of these properties that allow us to differentiate one relation from another.

Reflexive

A reflexive relation is a relation on a set where every element is related to itself. Formally, a relation R on a set A is reflexive if, for every a in A, the pair (a,a) is in R. In other words, for every element a in the set, the pair (a,a) belongs to the relation.

For example, let us consider Set A = {1,2,3} and a relation R = {(1,1), (2,2), (3,3)}. This relation is reflexive since for every element in A, A is related to itself.

Here, it is to be noted that, in order for a relation to be reflexive, it is not necessary to only have (a,a) pairs. For example, we can have a relation Z = {(x,x), (x,y), (y,y)} and this is still reflexive since the elements x and y are related to themselves.

Symmetric

A symmetric relation is a relation in which if the first element is connected to the second element, the second element is also related to the first element. Formally, a relation R on a set A is symmetric if, for every (a,b) pair in A, the pair (b,a) is also in A. In other words, to be symmetric, if the order of a pair were to be reversed, the reversed pair must be present.

For example, let us consider a relation R = {(1,2), (2,1), (3,4), (4,3)}. This relation is symmetric since for every (x,y) pair, we can also find a (y,x) pair.

Here, it is to be noted that, if there were no (x,y) pairs, then the relation will be considered symmetric by default. For example, a relation R = {(1,1), (2,2), (3,3)} is to be considered symmetric as the elements are only connected to themselves and not to any other element.

Transitive

a transitive relation is a type of relation on a set where if an element is related to another element and that second element is related to a third element, then the first element is also related to the third element. In other words, if (a,b) and (b,c) are in the relation, then (a,c) must also be in the relation.

For example, let us consider a relation R = {(1,2), (2,3), (1,3), (2,2)}. This relation is transitive since there is a (1,2) and (2,3) pair, and there is also a (1,3) pair. The simplest way to assess if a relation is transitive or not is this: look at a pair (let's say, (x,y)) and see if the second element of the pair (in this case y) is the first element of another pair; so, check if there is (y,z) or (y,b) pairs and so on; if such pairs exist, then check if first element of the original pair(in this case x) is related to the second element of the other pair (in the form of (x,z) or (x,b) in this case).

Here, it is to be noted that, if there were no (y,z) pairs or even (x,y) pairs, then the relation will be considered transitive by default. For example, a relation R = {(1,1), (2,2), (3,3), (6,7), (4,5)} is transitive since the second elements of the pairs are not the first elements of any other pair.

Closure of Relations

The closure of a relation is an extension of the original relation that includes additional pairs to satisfy certain properties. Sometimes, it may be necessary to add pairs to a set to make it reflexive, symmetric, and/or transitive.

Reflexive and Symmetric Closure

To perform reflexive closure, we simply add (a,a) pairs to the set for every element a in the domain that doesn't have (a,a) pair already. To perform symmetric closure, for every (x,y) pair, we simply add a (y,x) pair, if such pair is not present already.

Transitive Closure

Performing transitive closures are significantly more difficult because as, for every (x,y) and (y,z), we add a (x,z) pair to the relation, we may find a new set with a (y,z) pair that was not present initially. Therefore, we will have to add a new (x,z), which could continue this trend on its now. Therefore, transitive closures can be very iterative.

For example, let us consider relation R = {(1,2), (2,3), (3,1)}. Here, we observe the pairs (2,3) and (3,1), and to make this transitive, we add the pair (2,1). But as we add (2,1), we realize that we now have (1,2) and (2,1), and to make this transitive, we will have to add (1,1) to the relation. This complexity increases exponentially as the number of ordered pairs in a set increases.
